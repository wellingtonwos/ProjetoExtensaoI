# CarneUp 🥩

> **Sistema de Gestão para Açougues de Pequeno e Médio Porte.**

## 💻 Sobre o Projeto (teste)

O **CarneUp** é uma solução tecnológica desenvolvida como parte do **Projeto de Extensão I / Projeto Integrado** do curso de Análise e Desenvolvimento de Sistemas do **IFSP - Câmpus São Paulo**.

O objetivo do sistema é modernizar o "chão de loja" de açougues que ainda operam de forma analógica, oferecendo ferramentas para controle de estoque, gestão de validade de perecíveis e um PDV (Ponto de Venda) ágil integrado a balanças.

---

## ⚙️ Funcionalidades Principais

* **📦 Gestão de Estoque:** Controle de entrada e saída de produtos com rastreabilidade por lote.
* **🚨 Alerta de Validade:** Notificações visuais para produtos próximos ao vencimento (prevenção de perdas).
* **💰 Frente de Caixa (PDV):** Registro rápido de vendas.
* **📊 Relatórios:** Visão geral de vendas e movimentação.
* **🔐 Controle de Acesso:** Sistema de login seguro.

---

## 🛠 Tecnologias Utilizadas

### Back-end
* **Linguagem:** Java 17+
* **Framework:** Spring Boot 3
* **Segurança:** Spring Security
* **Banco de Dados:** PostgreSQL

### Front-end
* **Biblioteca:** React.js
* **Estilização:** CSS Modules / Styled Components
* **Consumo de API:** Axios / Fetch
